Simon


Written by Patrick Borgeat <patrick@borgeat.de>, http://www.cappel-nord.de


This is the game Simon implemented in SuperCollider. You could see it as a little hidden game in SuperCollider. Just repeat everything whatever Simon does by clicking on the buttons or use WASD on your Keyboard.


Just evaluate ...


Simon.says;


and have a lot of fun :)


You can also customize the Game. Here some Examples:


(


g = Simon.says;

g.waitTime_(0.5);

// How fast should Simon play?

g.waitDecrease_(0.05);

// How many Seconds should Simon get faster every turn

g.pauseTime_(2);

// How long should Simon wait after each Round?

g.setAllSynthDefs(\customSynthDef);

// Use a custom SynthDef for all Buttons. Default SynthDef is \simonDefault, provided by the class.

g.setSynthDef(\top, \customSynthDef);

// Just use the custom SynthDef for the top-Button. The Symbols for the four buttons are:

// \top, \bottom, \left, \right

g.setSynthArgs(\top, [\freq, 440]);

// Set the Synth-Arguments for each Button. 

)


Default Values: 


(

SynthDef(\simonDefault, {|out = 0, freq = 440, attack = 0.1, release = 0.7, pan = 0|

var env = EnvGen.kr(Env.perc(attack, release, 0.5), doneAction:2);

Out.ar(out, Pan2.ar(Blip.ar(freq,15,env),pan));

}).store;

g = Simon.says;

// this will actually overwrite the \simonDefault SynthDef

g.waitTime_(0.4);

g.pauseTime_(1.2);

g.waitDecrease_(0.01);

g.setAllSynthDefs(\simonDefault);

g.setSynthArgs(\bottom, [\freq, 60.midicps]);

g.setSynthArgs(\left, [\freq, 64.midicps, \pan, -0.4]);

g.setSynthArgs(\top, [\freq, 67.midicps]);

g.setSynthArgs(\right, [\freq, 72.midicps, \pan, 0.4]);

)
