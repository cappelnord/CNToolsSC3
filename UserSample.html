UserSample because I'm lazy!

Part of CNToolsSC3: http://github.com/cappelnord/CNToolsSC3


superclass: Object


Most of the time you have all your Samples in one Folder. Always writing the same path to get to my samples was annoying. So i wrote a little class with some extra Functionality. The class can't be instantiated, it has only static methods and members. Calling new will redirect to Buffer.read and will return an Buffer. If the sample was allready loaded by UserSample it will somply return the Buffer. Don't forget to boot your Default Server.


You have to edit your default sample path, either by editing the Sourcecode or by setting UserSample.base in your startup file.


If you have to restart your Server (but not the language) you have to do UserSample.freeAll


Check out some methods:


*new(path)

Returns a Buffer Object. If this is the first time it will load the sample into a buffer.

*at(path)

Returns the Buffer for the given path (if available).

*list

Lists all Bufers.

*free(path)

Frees the Buffer at the specified path.

*freeAll

Frees everything.Â
